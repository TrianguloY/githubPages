This is an encrypter for blogger, but can be used elsewhere. The idea is that a part of a page is encrypted by a password, and can only be decrypted with that same password. No password, no page.
The funny thing, is that the encrypter was encrypted by itself, and this below was the result.

If you want to access the encrypter just solve this simple riddle: which number has the same number of letters (in english) as itself? The answer is a single digit...
